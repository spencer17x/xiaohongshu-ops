# Image Sources

## Allowed source policy

Use only watermark-free sources.
Do not remove watermarks.
If a source adds visible watermarks or unclear licensing, reject it.

## Preferred source order

1. Unsplash
2. Pexels
3. Pixabay

## Topic mapping notes

When the post topic is abstract (for example AI, workflow, productivity), prefer images matching one of these concepts:
- laptop workspace
- person working at desk
- digital interface
- notebook / planning / focus
- keyboard / monitor / coding / analysis
- office desk with clean composition

Avoid obviously cheesy marketing visuals.
Avoid fake dashboards with giant promo text.
Prefer calm, editorial, modern images.

## Mixed mode rule

- First try real stock images
- If relevant images are weak or too generic, fall back to lightweight cards
- Do not generate text-heavy cards unless explicitly requested
