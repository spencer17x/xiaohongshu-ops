Use this exact high-signal structure when the user asks for a Xiaohongshu post draft.

- 主标题：
- 副标题：

#### 第2页
- 标题：
- 要点：
- 要点：

#### 第3页
- 标题：

- 要点：

#### 第4页

- 要点：

(如果有第5/6页继续补)

### 正文

### 标签
- #标签1
- #标签2
- #标签3
- #标签4
