# Stock Providers

## Unsplash

- Good for modern editorial workspace images
- Requires API integration for reliable search/download automation at scale
- Prefer landscape/portrait workspace photos with clean composition

## Pexels

- Strong for office, laptop, desk, productivity scenes
- API-friendly provider when a key is configured

## Pixabay

- Broad free-image coverage
- Useful as a fallback source

## Provider policy

Preferred order:
1. Unsplash
2. Pexels
3. Pixabay

If no provider credentials are configured yet, use the query-planning script first and keep the stock-image step in planning mode rather than pretending downloads already exist.
