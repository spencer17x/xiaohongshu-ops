# Auto Learning

## Goal

Learn from recent Xiaohongshu-relevant trends and style patterns on a daily basis, then store the result as local reference material for future content generation.

## Default behavior

- enabled: true
- frequency: daily
- mode: reference-only
- updateProfileDirectly: false

## What to learn

- topic heat and recurring angles
- title phrasing patterns
- high-signal structure ideas
- language patterns worth borrowing conceptually
- low-quality patterns to avoid

## What not to do

- do not copy posts verbatim
- do not plagiarize structure line-by-line
- do not remove watermarks from sourced content
- do not silently rewrite the main profile without explicit user approval

## Suggested local outputs

- `data/xiaohongshu/learning.json`
- `data/xiaohongshu/trends.json`
- `data/xiaohongshu/style-notes.md`

These are reference files, not the profile itself.
