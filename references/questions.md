# Setup Questions

Use these when initializing Xiaohongshu operations. Ask them in one compact message.

1. 你的小红书账号主要想做什么方向？
2. 你做这个号最主要的目标是什么？（吸粉 / 转化 / 人设 / 测试）
3. 你希望整体风格更偏：干货、观点、教程，还是带一点焦虑感的增长型内容？
4. 你主要想吸引哪类人？
5. 默认每篇想做几页？（2-6 页）
6. 要不要默认生成图文卡片？
7. 发布前是否一律人工确认？

If the user says “你帮我定”, prefer:
- AI / 效率 / 工作流
- 目标：吸粉
- 风格：干货 + 观点 + 适度焦虑
- 受众：职场人 / AI 初学者
- 页数：4
- 默认生成图文卡片：是
- 发布前人工确认：否（默认全自动发布）
